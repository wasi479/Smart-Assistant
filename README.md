# Smart Assistant - Android App

An AI-powered Android assistant application built with Java and Android Studio.

## Features
- AI conversational responses via OpenAI API
- Location-based navigation using Google Maps SDK
- Firebase Authentication and Cloud Firestore for user management
- Push notifications and real-time alerts
- Android sensor integration for context-aware functionality
- Notes and Tasks management

## Tech Stack
| Category | Technologies |
|----------|-------------|
| Language | Java |
| IDE | Android Studio |
| AI | OpenAI API / Gemini API |
| Backend | Firebase Auth, Cloud Firestore |
| Maps | Google Maps SDK |
| UI | RecyclerView, Navigation Components |

## Project Structure
- `AIFragment` - AI chat interface
- `HomeFragment` - Main dashboard
- `NotesFragment` - Notes management
- `TasksFragment` - Task management
- `ProfileFragment` - User profile
- `LoginActivity / SignupActivity` - Authentication
